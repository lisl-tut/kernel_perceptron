# kernel_perceptron
カーネルパーセプトロンに関してのプログラム． 
カーネルパーセプトロンが境界面を更新する状況を描画できる．

# 環境
必要なライブラリと動作確認済みのバージョンは以下の通り 
- python 3.7.3
- numpy 1.16.4  
- matplotlib 3.1.0

# 使い方
execute.shの中身を変更してプログラムの設定を行い，実行するだけ．
